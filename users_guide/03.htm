Preferences

When you first run the program, you will be prompted to enter the name by which you would like to be known to other players. Enter a name a press OK. (You may change your name later in your Preferences settings.)

Next, you will be asked to enter a password. Enter a password and press OK.

In the VASL Controls window, set your Preferences by going to File > Edit Preferences. Most important is the Board Directory preference to inform VASL where to find the board images.

Personal

Name

Whatever you enter in the Name field is the name that other people will see in the Chat area of VASL. Use a nickname or put your full name. Its up to you.

Personal Info

Enter any personal info which you would like to make available to other VASL users.

You may read another player's profile by right-clicking on their name in the Server window and selecting Show Profile.

General

Board Directory

Click the Select button and select the directory that contains the board files you have downloaded from the VASL Board Archive. (If this setting is incorrect, no boards will appear when you open a game.)

Use Arrow Keys to Scroll

Checking this preference allows you to use the arow keys on your keyboard to scroll around the VASL map when you have no counters selected. If this box is not selected, the number keyboard will scroll the map window.

Auto-Report Moves

The Auto-report moves checkbox will echo your units' moves into the VASL chat window so that you don't have to move a unit from E5 to E4 and then type in "E5 to E4"; VASL will detect the move and auto-report it for you. Covered-arc changes are reported as well.

Center on Opponent's Move

The Center on Opponent's Move checkbox will automatically center the map on a unit that is being moved, whether connected to the server or reading a logfile.

Marked Moved Units

The Mark Moved Units checkbox will automatically mark units after they have been moved. Click Mark Unmoved at the top of the VASL Map window to clear the "Moved" status of all units.

Chat Window Font

Specifies the font and font size used in the VASL Controls window.

LOS

Retain LOS-hindrance counters

Checking this preference will cause all counters which could cause a LOS hindrance to remain on the board when a LOS check is made.

Snap Thread to grid

The Snap Thread to grid checkbox will automatically place the LOS thread at the nearest hex center or vertex. Click the LOS button at the top of the VASL Map window to check LOS by dragging the mouse.

Thread Color / Hindrance Thread Color / Blocked Thread Color

Allows you to choose the color that the LOS Thread appears in.

Nationality Colors

Allows you to customize the color of each nationality's units. Use the swatch, HSB or RGB color pallettes to select the desired color.

Click Default to return the color to the default setting.

 

Editing Preferences

Occasionally there will come a time where you will want to edit the preferences. To edit your preferences, launch VASL and choose File > Edit Preferences from the menu at the top of the VASL Controls window.
